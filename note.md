## os

os.arch() : cpu 的架构名

**os.cpus()** : cpu 的内核

os.freemem() : 可用内存

**os.tmpdir()** : 操作系统的临时目录

## path

path中的所有方法都不会检查路径是否存在，只是根据规则来处理

path.sep(分隔符):

- windows \
- linux  /

node跨平台

path.resolve(./a.js) : ./相对的是命令行(process.cwd()) 

## url

分析url，做字符串处理

host: 带端口号
hostname: 不带端口号
